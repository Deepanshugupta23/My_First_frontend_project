This is a summary of my Quiz Plateform- My_Day.
I am Deepanshu Gupta, and I developed a frontend React project called “My_Day”  Which is a Quiz Platform website that allows users to create, manage, and attempt quizzes in an interactive way. This project is built using React.js, React Router, JavaScript, and Tailwind CSS, focusing on component-based architecture and a smooth user experience.
The platform starts with a Home Page, which serves as the main entry point of the website. From here, users can navigate to different sections of the application. One of the core features is the Create New Quiz page, where users can design their own quizzes by selecting different question types such as MCQ (Single Correct), MCQ (Multiple Correct), Short Answer questions, and Description-based questions. The system allows dynamic question creation, option management, and validation before saving the quiz.
All created quizzes are stored and displayed on the My Quiz page in a structured table format. This page provides full quiz management functionality, including viewing quiz details, editing quiz titles, activating or deactivating quizzes, and deleting quizzes. This makes it easy for users to organize and control the quizzes they have created.
The Play Quiz section allows users to attempt active quizzes. Before starting, users enter their name for basic authentication, which personalizes the quiz experience. The platform supports multiple question formats during gameplay and tracks the user’s answers in real time.
After completing a quiz, users are redirected to the Result Page, where they can see their performance, including total questions, correct answers, and wrong answers. The application uses localStorage to manage quiz data and results, enabling persistent data storage without requiring a backend.
Overall, the My_Day Quiz Platform demonstrates strong frontend development skills, including React state management, dynamic form handling, routing, data persistence, and interactive UI design, making it a practical and user-friendly quiz management system.
